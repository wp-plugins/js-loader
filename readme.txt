== Description ==

A collections of Javascript libraries that often used by developers. Easily enable and disable them from the admin panel.

For more information, check out on [Github](https://github.com/calibreworks/js_loader).

== Installation ==

This section describes how to install the plugin and get it working.

1. Find `js_loader` plugin or upload `js_loader` folder to the `/wp-content/plugins/` directory 
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > JS Loader, you will find several javascript libraries there, enable any library you want
